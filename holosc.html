Side projects

HoloSC is a program actively used in BFK Polska. It's purpose is to improve document flow across company. Delivery/issue points have client module installed on their computer that communicates with truck scale through COM interface. Data is send through VPN to headquarters where it's inserted into MySQL database.

Browser module enables analysts to check inserted data by date and facility. It also has a simple report system that calculates total weights and validates data.

System requires users to be authenticated in order to authorize and keep track of their actions. Credentials are stored in MySQL database (hashed with salt).

Written in WPF using MVVM pattern.
