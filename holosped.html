HoloSped is currently in development.

It's meant to improve general workflow in transport companies (spedition sector). Planned features:

  • Task management - creating tasks for users (ex. drivers) with customized data
  • Reminder system - cyclic tasks/personal notes
  • Communicator - chat system adapted to spedition needs

Whole system will consist of:

  • Web Application - created in Blazor
  • Android app - created in Xamarin
  • WEB API - data hub for all clients; will handle all requests in system
  • MySQL database to store data
