HoloFish

Simple Xamarin project for fish cataloguing. Using RFID reader or android keyboard user can input fish ID to open its card. Main card contains general informations and list of control entries - to track physical changes of a fish in time (length, weight, appearance - photo). It uses SQLite as database.

Drax

Client-Server system for IT infrastrucure managment considering printers and its consumable components. It's supposed to live monitor each network printer via its API and report low levels. Additionally system had an inventory system to organize data. It was built using WPF, Windows Service and MySQL. Development was dropped because of more urgent cases followed by change of departments.
