Side projects

HoloYT started as a tutorial project. I wanted to get a grasp of how multithreading in .NET works (turned out that many inner frameworks handle multithreading differently). While working for TVP I noticed that common problem for journalists was acquiring media from internet and uploading it to our inner system. I came up with an idea for a "single click" tool that's easy to use for everyone. Instead of writing an email to IT department with Youtube link, user would be able to paste it directly to HoloYT and it will handle the rest. Multithreading enabled to handle multiple links at the same time speeding up the whole process.

In its core it's a wrapper for a commandline tool "youtube-dl" which is no longer supported. I also stopped supporting it after leaving my job at TVP. Quick research shows that there is another popular commandline tool which is a fork of "youtube-dl". Maybe in future I will adjust HoloYT to work with this new tool.
